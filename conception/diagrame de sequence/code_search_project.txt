API -> DB: Build MongoDB query
note right of API
  - Full-text search on title, description
  - Filter by category, riskLevel, fundingGoal range

  - Sort by relevance, creation date, or funding progress
end note
API -> DB: db.projects.find({ $text: { $search: query }, status: "ACTIVE", ... }).sort({ createdAt: -1 }).limit(20)
DB --> API: projects
