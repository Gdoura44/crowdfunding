// ============================================================================
// AI ANALYSIS WORKFLOW & RETRY LOGIC
// ============================================================================
//
// **Status Flow:**
// DRAFT → (user submits) → AWAITING_AI → (AI success) → UNDER_REVIEW
// AWAITING_AI → (AI fails) → aiStatus=FAILED, status remains AWAITING_AI
// SUSPENDED → UNDER_REVIEW (admin reactivates project and AI analysis is still fresh)
// SUSPENDED → AWAITING_AI (admin reactivates project but AI analysis is stale: 
// analyzedAt< now - 30 days)
//
// **Retry Mechanism (Admin Action):**
// 1. Admin clicks "Retry Failed Analysis" (UC_RetryAnalysis)
// 2. aiAnalysisRetries incremented (max 3)
// 3. New AI job pushed to BullMQ
// 4. aiStatus reset to PENDING
//
// **Retry Exhaustion (Automatic):**
// - After 3 manual retries exhausted (aiAnalysisRetries >= 3):
//     • aiStatus remains FAILED
//     • Further retries are blocked (admin can still review manually)
//     • Project.status stays AWAITING_AI (no automatic transition to DRAFT)
//     • Admin notification created: "Project {title} AI analysis failed after 3 retries"
//     • Creator notification: "Your project requires manual review"
//
// **State Transitions:**
// aiStatus: PENDING → COMPLETED (success)
// aiStatus: PENDING → FAILED (n8n workflow fails)
// aiStatus: FAILED → PENDING (admin retries, retries < 3)
// aiStatus: FAILED → FAILED (retries exhausted, manual admin intervention required)
//
// **Retry Precondition:**
// Admin can retry AI analysis only when:
// - Project.status = AWAITING_AI
// - Project.aiStatus = FAILED
// - Project.aiAnalysisRetries < 3
//

//

// can be archived.
// Archived projects are hidden from public search but visible on creator dashboard.
// Projects with status ACTIVE, APPROVED, CLOSED, or SUSPENDED cannot be archived.
//
// When a project is archived:
//   - isArchived = true
//   - status remains unchanged
//   - No further edits allowed (except unarchive by admin)
//
// Unarchiving resets isArchived = false, status remains as is.
