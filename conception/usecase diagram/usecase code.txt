  ' --- Notes attached to specific use cases ---
  note right of UC_Login
  **Authentication Requirements**  

  **Public (no authentication required):**  
  • Register  
  • Login  
  • Resend Verification Email  
  • Reset Password  
  • View Project (read-only)  
  • Search Projects (read-only)  

  **Authenticated Users (valid JWT required):**  
  All other use cases require a valid JWT token, including:  
  • Logout  
  • Update Profile, Delete Account  
  • Create/Edit/Archive/Resubmit Project  
  • Invest / Cancel Investment / View Investments  
  • Report Project  
  • View Notifications, View Payouts, Provide Bank Details  
  • Get Recommendations, View Risk Score, View AI Analysis  
end note

  note right of UC_ValidateProject
    **Admin Only (JWT + role=ADMIN):**  
    • Validate Project, Publish Project, Reactivate Project  
    • View Reports, Resolve Report  
    • Manage Users, Manage Notifications  
    • Approve Payouts  
    • Retry Failed Analysis/Cancellation/Refund/Payout/Notification  
  end note

  note right of UC_DeleteAccount
    **Deletion Restrictions:**  
    Users cannot delete account if they have:  
    • Active investments (status: INITIATED or CANCELLING)  
    • Pending/Ready payouts (status: PENDING or READY)
    • Any project they created with status = FUNDED (goal reached but not closed) 
  end note

  note right of UC_Chatbot
  **Chatbot for investors:**  
  - Available to authenticated users.  
  - Rate limited to 10 questions per hour.  
  - Answers based on project data and platform rules.  
  - Does not give financial advice.  
  - Direct call to Gemini API (no n8n).
  end note
}
